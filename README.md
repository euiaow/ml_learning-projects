# Учебные проекты

Подборка моих учебных работ по ML, алгоритмам и математике. Основная цель — разобраться в алгоритмах «руками» и применить математику на практике.

## Что здесь есть
- **ML-методы**: реализация One-Hot Encoding, линейной регрессии с нуля.
- **Анализ данных**: разведывательный анализ датасетов (Iris).
- **Алгоритмы**: обходы графов, поиск кратчайших путей.

## Инструменты
Python, NumPy, Pandas, Matplotlib, Jupyter Notebook.

## Как запустить
Все ноутбуки самодостаточны. Установите зависимости из `requirements.txt` и откройте любой `.ipynb` файл в Jupyter.

## Важно
Все проекты — учебные. Их цель не production-качество, а демонстрация понимания базовых концепций.
